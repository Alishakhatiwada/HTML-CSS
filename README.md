# HTML Laboratory Assignment

## Description
This repository contains the completed HTML Laboratory assignment.
It covers HTML fundamentals, text formatting, lists, images, forms,
tables, semantic HTML5 elements, accessibility, and basic SEO concepts.
The project is designed to build a strong foundation in HTML and prepare
for CSS and JavaScript.

## Project Structure
- HTML Exercise/ → Exercise-wise HTML files
- htm-lab/ → Module-wise HTML laboratory files
- assets/ → Images and other resources
- README.md → Project documentation

## How to Run the Project
1. Clone the repository:
   git clone https://github.com/Alishakhatiwada/HTML-CSS.git

2. Open the project folder.

3. Open any `.html` file in a web browser
   (Right click → Open with → Chrome / Edge / Firefox).

No additional setup or dependencies are required.
